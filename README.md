# Лабораторная работа №10-11. Создание приложения «Список студентов» с использованием Material Design

Приложение для отображения списка студентов с возможностью раскрытия подробной информации о каждом. 

## Цель работы

Научиться применять принципы Material Design в Androidприложениях с использованием Jetpack Compose. В результате лабораторной работы будет создано приложение «Список студентов», которое демонстрирует использование цветовых схем, типографики, форм и анимаций Material Design.

## Функциональность

- Отображение списка студентов с аватарками
- Поддержка светлой и тёмной тем оформления
- Динамическое переключение темы (следует за системными настройками)
- Круглые аватарки студентов с правильным кадрированием изображений
- Отображение имени и возраста студента в каждой карточке
- Разворачивающиеся карточки с подробным описанием студента
- Плавная анимация раскрытия/сворачивания карточек (пружинная анимация)
- Визуальная обратная связь при взаимодействии (изменение иконки кнопки)
- Адаптивный интерфейс с использованием Scaffold и TopAppBar

## Технологии

- **Язык программирования**: Kotlin
- **UI Toolkit**: Jetpack Compose
- **Дизайн-система**: Material Design 3
- **Компоненты Material 3**:
  - Card — для карточек студентов
  - CenterAlignedTopAppBar — для верхней панели приложения
  - LazyColumn — для эффективного отображения списка
  - Scaffold — для базовой структуры экрана
- **Шрифты**: 
  - Montserrat (Google Fonts) — для основного текста
  - Abril Fatface (Google Fonts) — для заголовков
- **Анимации**: Spring Animation (пружинная анимация) с настраиваемыми параметрами затухания и жёсткости
- **Иконки**: Material Icons Extended
- **Система контроля версий**: Git, GitHub
